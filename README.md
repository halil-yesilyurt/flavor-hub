## üì• Installation

1. Clone the repository:
```bash
git clone https://github.com/halil-yesilyurt/flavor-hub.git
```

2. Navigate to project directory:
```bash
cd flavor-hub
```

3. Open index.html in your browser or use a local server:
```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve
```

## üöÄ Usage
